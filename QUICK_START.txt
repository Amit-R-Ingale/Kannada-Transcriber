========================================================================
HOW TO USE AFTER INSTALLATION
========================================================================

  1. Double-click the desktop icon
  2. Browser opens automatically at http://localhost:8000
  3. Click microphone button
  4. Grant permission when browser asks
  5. Speak into your microphone
  6. See text appear in real-time!
  7. Use export buttons to save as TXT or Word

========================================================================
EXPORT OPTIONS
========================================================================

  TXT - Plain text file
  Word - Microsoft Word document with Nudi font support
  Copy - Copy to clipboard

========================================================================
TROUBLESHOOTING
========================================================================

  Microphone not working?
    Check browser permissions (click lock icon in address bar)
    Make sure microphone is connected
    Close other apps using microphone

  Browser not opening?
    Manually open: http://localhost:8000
    Try Chrome or Edge browser
    Check Windows Firewall settings

  App not starting?
    Right-click icon > Run as administrator
    Check antivirus isn't blocking it
    Reinstall the application

========================================================================
UNINSTALLATION
========================================================================

  To uninstall:
    Windows Settings > Apps > Apps & Features
    Find "Free Online Transcription"
    Click Uninstall
