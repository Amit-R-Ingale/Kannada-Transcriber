# Free Online Transcription - Quick Start Guide

== Getting Started (3 Easy Steps) ==

Method 1: Using Portable Version (No Installation)
1. Extract the "Free_Online_Transcription_Portable" folder
2. Double-click "Start.bat"
3. Browser opens automatically - Start transcribing!

Method 2: Using Installer
1. Run "Free_Online_Transcription_Complete_Setup_v1.0.exe"
2. Follow installation wizard
3. Launch from Start Menu or Desktop

== Main Features ==

### 1. Real-Time Transcription
- Click the microphone button
- Speak clearly into your microphone
- See text appear in real-time
- Click stop when done

### 2. Audio File Upload
- Switch to "Upload Audio File" tab
- Drag and drop audio file OR click "Browse Files"
- Supports: MP3, WAV, M4A, OGG, FLAC
- Automatic transcription starts
- Works with files up to 30MB

### 3. Language Support
- 99+ languages supported
- Select your language from dropdown
- Auto-detect available for real-time

## üîß Troubleshooting

### Microphone Issues
**Problem**: Microphone not detected
**Solution**:
1. Check if microphone is connected
2. Allow microphone permission in browser
3. Check Windows settings:
   - Settings ‚Üí Privacy & Security ‚Üí Microphone
   - Enable for your browser

**Problem**: "Permission denied" error
**Solution**:
1. Click the lock icon in browser address bar
2. Find "Microphone"
3. Change to "Allow"
4. Reload page (F5)

### File Upload Issues
**Problem**: File upload not working
**Solution**:
1. Check file format (MP3, WAV, M4A, OGG, FLAC)
2. Ensure file is under 30MB
3. Make sure FFmpeg is installed/bundled
4. For installed version: Reinstall to get FFmpeg

**Problem**: "Transcription failed" error
**Solution**:
1. Try a shorter audio file (under 5 minutes works best)
2. Convert to WAV format
3. Reduce audio quality to make file smaller
4. Use real-time recording instead

### History Not Saving
**Problem**: History disappears after closing
**Solution**:
1. Check 'data' folder exists (portable version)
2. For installed version: Check folder permissions
3. Try running as Administrator
4. Check browser localStorage is enabled

### Port Already in Use
**Problem**: "Port 8000 already in use"
**Solution**:
1. Close any other instance of the app
2. Close other apps using port 8000
3. Restart computer if needed

## üí° Tips for Best Results

### For Real-Time Transcription:
- Speak clearly and at moderate speed
- Use a good quality microphone
- Minimize background noise
- Pause briefly between sentences
- Review and edit afterwards

### For File Upload:
- Use good quality audio recordings
- Keep files under 5 minutes for best results
- WAV format works best
- Reduce background noise before recording
- Split long recordings into segments

### For Indian Languages:
- Select the correct language before starting
- Speak clearly in the selected language
- Don't mix languages in one sentence
- Use the Word export for proper font rendering
